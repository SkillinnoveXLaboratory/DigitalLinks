Important:
- Output plain text only.
- Do not use Markdown.
- Do not use #, ##, ### headings.
- Do not use bold, italics, **, *, bullets, tables, or code blocks.
- Do not use horizontal lines like ---.
- Do not add notes, comments, suggestions, or explanations before or after the quotation.
- Use only normal plain text lines.
- Use normal hyphen "-" only, not long dash.
- Use "Rs" instead of the rupee symbol to avoid encoding problems.
- Put every feature, module, platform, charge, and exclusion on its own line.
- Keep section headings exactly as written below.
- Keep actor labels exactly like: Users can:, Vendors can:, Admin can:, Website can:
- Keep technology stack in Label: Value format only.
- Keep cost in one exact line like: Total Development Cost: Rs 50000
- Keep timeline in plain text only.
